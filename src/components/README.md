# components/

Reusable UI pieces.

Small, focused components composed together by the pages.

Typical files:
- `StateBar.tsx` — shows current agent state, dropdown to change it
- `ContactCard.tsx` — appears when a contact is offered (Accept / Reject)
- `CallControls.tsx` — mute, hold, transfer, end
- `CustomerPanel.tsx` — screen-pop / customer data
